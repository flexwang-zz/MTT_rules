## 重买（Rebuy)
前三个盲注级别（比赛开始大约两小时）可以至多重买一次（50¥）。重买花费进入奖池。
## 钱圈
钱圈一共4名玩家，将瓜分奖池。比例分配如下：

| 名次 | 奖金比例 |
|------|----------|
| 1    | 40%      |
| 2    | 30%      |
| 3    | 20%      |
| 4    | 10%      |
